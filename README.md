# Gymnasium Examples
Some simple examples of Gymnasium environments and wrappers.
For some explanations of these examples, see the [Gymnasium documentation](https://gymnasium.farama.org).

### Environments
This repository hosts the example of manipulating KUKA iiwa 14 using Reinforcement Learning in MuJoCo physics simulator.

```{shell}
cd gymnasium_env
pip install -e .
```
